# Guide de démarrage — membres du Conseil Syndical

> Registre des décisions du CS — **https://cs-rives.vercel.app**
> Version Word à partager : `docs/Guide_demarrage_CS.docx`. À jour au 2026-07-18.

---

## 1. Première connexion

Vous avez reçu du président votre **email** et un **mot de passe provisoire**.

1. Ouvrir un navigateur et aller sur **cs-rives.vercel.app**.
2. Entrer votre **email**.
3. Entrer le **mot de passe provisoire** communiqué par le président.
4. Cliquer sur **« Se connecter »**.
5. Sur l’écran **« Bienvenue »**, entrer un **nouveau mot de passe** (au moins 8 caractères).
6. Le retaper dans **« Confirmer le mot de passe »**.
7. Cliquer sur **« Définir mon mot de passe »**.

*Vous arrivez alors sur le registre des décisions.*

---

## 2. Mot de passe oublié

1. Sur la page de connexion, cliquer sur **« Mot de passe oublié ? »**.
2. Entrer votre **email** et valider.
3. Ouvrir le lien reçu par email.
4. Définir un **nouveau mot de passe**.

---

## 3. Voter sur téléphone

1. Aller sur **cs-rives.vercel.app** et se connecter.
2. Le registre s’affiche en **liste de cartes**.
3. Repérer une décision portant la pastille **« à voter »** (la carte est rouge si le délai est dépassé).
4. Toucher la **carte** de la décision.
5. Lire l’objet, la description et les pièces jointes (toucher un devis pour l’ouvrir).
6. Dans le bloc **« Votre vote »**, toucher **Pour**, **Contre** ou **Abstention**.
7. Le vote est **enregistré immédiatement**.

*Vous pouvez changer votre vote tant que le président n’a pas enregistré la décision. Un commentaire est possible sous les boutons.*

---

## 4. Voter sur ordinateur

1. Aller sur **cs-rives.vercel.app** et se connecter.
2. Le registre s’affiche en **tableau**.
3. Cliquer sur le **titre** de la décision.
4. Lire l’objet, la description et les pièces jointes (cliquer un devis pour l’ouvrir).
5. Dans le bloc **« Votre vote »**, cliquer **Pour**, **Contre** ou **Abstention**.
6. Le vote est **enregistré immédiatement**.

---

## 5. Créer une décision (sur ordinateur)

1. Sur le registre, cliquer sur **« + Nouvelle décision »** (en haut à droite).
2. Entrer le **titre** de la décision (obligatoire).
3. Rédiger la **description** (facultatif).
4. Si la décision engage de l’argent : choisir le **projet** ou la **résolution** concernée.
5. Entrer le **montant engagé** (ex : 20’000).
6. Ajouter les devis : cliquer **« + Ajouter un fichier »**.
7. Cliquer sur **« Créer la décision »**.
8. Cliquer sur **« Prévenir le CS »** pour avertir les membres (ouvre WhatsApp).

*La création se fait uniquement sur ordinateur. Ensuite les membres votent, puis le président enregistre la décision.*
